# 🤖 Automated AI ATS & Culture-Fit Screener

A weekend project to build an end-to-end automated Applicant Tracking System (ATS) powered by Python and Google's Gemini AI. 

## 💡 The Problem
Traditional recruitment involves scanning hundreds of CVs manually. Many AI screeners today can easily be tricked by candidates using ChatGPT to write their cover letters. 

## 🚀 The Solution (Anti-Cheat ATS)
This bot acts as a "strict HR Manager". It doesn't just read the applicant's story; it actively extracts text from their uploaded PDF CV and cross-references the claims. If a candidate writes a beautiful story but has an empty or mismatched CV, the AI will heavily penalize their score.

## ✨ Features
1. **Auto-Data Fetching:** Pulls applicant data (Form submissions) directly from Google Sheets API.
2. **PDF Extraction:** Downloads and extracts raw text from PDF CVs hosted on Google Drive.
3. **AI Assessment (Gemini 2.5 Flash):** Evaluates Culture-Fit, Proactivity, and Problem-Solving skills using strict conditional prompts (Temperature = 0.0 for reliability).
4. **Automated Decision Emails:** Sends real-time acceptance or rejection emails to candidates via SMTP, including constructive feedback generated by the AI.

## 🛠️ Tech Stack
* **Language:** Python 3
* **APIs & Libraries:** `google-genai` (Gemini), `gspread` (Google Sheets), `PyPDF2` (PDF parsing), `smtplib` (Email automation).

## ⚠️ Disclaimer
This is a personal learning project built over a weekend. The architecture, error handling, and prompt engineering are continuously being improved. The uploaded `main.py` is a template; real API keys and credentials have been removed for security purposes.
